# Data Structure And Algorithm Assignment
Make own data structure to prove that 2^67-1 have divisors
1. How to store large number in variable
2. How to calculate it

# Solution
I think that Splitting into digits can make solve this problem.
First, 
```C
int power[21] = {0,};

power[0] = 1;
for (int i = 0; i < 67; i++) {
  for (int j = 0; j < 21; j++) { // 제곱 연산
    power[j] *= 2;
  }
  for (int j = 0; j < 21; j++) { // 자리 올림
    if (power[j] >= 10) {
      power[j + 1] += power[j] / 10;
      power[j] -= (power[j] / 10) * 10;
    }
  }
}
power[0] -= 1;
```
```
power = {7, 2, 9, 2, 1, 4, 6, 7, 6, 9, 8, 5, 2, 5, 9, 3, 7, 5, 7, 4, 1}
```
Second,
```C
int multiply[21] = {0,};
int divisorA[12] = {7, 8, 2, 7, 5, 2, 8, 3, 8, 1, 6, 7};
int divisorB[9] = {1, 2, 7, 7, 0, 7, 3, 9, 1};

for (int i = 0; i < 9; i++) {    // divisorB[9]
  for (int j = 0; j < 12; j++) { // divisorA[12]
    multiply[i + j] += divisorA[j] * divisorB[i]; // 곱셈 연산
  }
}
for (int i = 0; i < 21; i++) { // 자리 올림
  if (multiply[i] / 10 != 0) {
    multiply[i + 1] += multiply[i] / 10;
    multiply[i] -= multiply[i] / 10 * 10;
  }
}
```
```
multiply = {7, 2, 9, 2, 1, 4, 6, 7, 6, 9, 8, 5, 2, 5, 9, 3, 7, 5, 7, 4, 1}
```
